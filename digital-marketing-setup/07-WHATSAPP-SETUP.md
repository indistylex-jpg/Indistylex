# 07 — WhatsApp Business Setup

## What This Does
Sets up WhatsApp Business for customer support, order updates, marketing broadcasts, and catalog sharing.

**Time needed**: 30 minutes

---

## Step 1: Install WhatsApp Business App

1. Download **WhatsApp Business** (NOT regular WhatsApp) from Play Store/App Store
2. Register with your **business phone number**
   - Use a dedicated number (not personal)
   - Can use landline (they'll call with code)
3. Verify via SMS or call

> ⚠️ You cannot use same number for WhatsApp + WhatsApp Business simultaneously.

---

## Step 2: Set Up Business Profile

1. Go to **Settings** → **Business Settings** → **Profile**
2. Fill everything:

| Field | Value |
|-------|-------|
| Business Name | `Indistylex - Kids Fashion` |
| Category | `Clothing & Apparel` |
| Description | (see below) |
| Address | Your business address |
| Business Hours | Mon-Sun 10AM-9:30PM |
| Email | `hello@indistylex.com` |
| Website | `https://indistylex.com` |

### Business Description (256 characters):
```
Premium kids clothing brand for 0-14 years. Stylish, comfortable & affordable fashion. 
Free shipping across India. COD available. Easy 7-day returns.
🛍️ Shop: indistylex.com
```

### Profile Photo:
- Use your square logo (clear at small size)

### Cover Photo:
- Use a product collection image or lifestyle shot

## Step 3: Set Up Catalog

### 3.1 Add Products
1. Go to **Business Settings** → **Catalog**
2. Click **"Add New Item"**
3. For each product:

| Field | Example |
|-------|---------|
| Name | Boys Cotton T-Shirt - Blue |
| Price | ₹499 |
| Description | Soft 100% cotton, ages 2-8 years. Available in S/M/L. |
| Link | https://indistylex.com/product/boys-tshirt-blue |
| Product Code | SKU-BT-001 |
| Images | Up to 10 photos per item |

### 3.2 Organize by Collections
Create collections:
- 🆕 New Arrivals
- 👦 Boys Collection
- 👧 Girls Collection
- 🎉 Party Wear
- 🏷️ Sale Items
- 🎊 Festive Collection

> Add your **top 20-30 products** to start. Update weekly.

## Step 6: WhatsApp Marketing Broadcasts

### 6.1 Create Broadcast Lists
1. Go to **Broadcasts** → **New Broadcast**
2. Create lists:
   - All Customers (past buyers)
   - Interested (inquired but didn't buy)
   - VIP (repeat buyers, high spenders)

### 6.2 Broadcast Templates

**New Arrival:**
```

[Product Name] — just dropped! 

✨ [Key feature 1]
✨ [Key feature 2]
💰 Only ₹[price]

🛍️ Shop now: [product link]

Limited stock — first come, first served! 

Reply STOP to unsubscribe.
```

**Order Update (Personal):**

### 6.3 Broadcast Rules:
- ⚠️ Only send to people who have your number saved (otherwise won't receive)
- Maximum 256 contacts per broadcast list
- Send maximum 2-3 broadcasts per week
- Always include "Reply STOP to unsubscribe"
- Best times: 11 AM or 7 PM IST
- Don't send daily — people will block you

## Step 7: WhatsApp Business API (Advanced — Later)

### When to Upgrade:
- More than 256 contacts to message
- Need automated notifications (order updates)
- Want chatbot capabilities
- Need green tick verification

### API Providers in India:
| Provider | Starting Cost | Best For |
|----------|--------------|----------|
| Interakt | ₹999/month | Small businesses |
| WATI | ₹2,499/month | Medium businesses |
| AiSensy | ₹999/month | Budget option |
| Gallabox | ₹1,199/month | E-commerce focus |

### What API Enables:
- Unlimited broadcast messages
- Automated order confirmations
- Abandoned cart reminders on WhatsApp
- Chatbot with product catalog
- Green tick verified badge
- Multi-agent support (team)

> Start with free WhatsApp Business app. Upgrade to API when you have 500+ customers.

## Step 8: WhatsApp Click-to-Chat Link

### Create Your Link:
```
https://wa.me/91XXXXXXXXXX?text=Hi!%20I'm%20interested%20in%20Indistylex%20kids%20clothing
```

Replace `91XXXXXXXXXX` with your actual number (with country code, no +).

### Where to Use This Link:
- Instagram bio
- Facebook page CTA button
- Website floating WhatsApp button
- Email signatures
- Google Business profile
- Product pages ("Ask about this product")

## Step 9: Order Management via WhatsApp

### Order Flow:
1. Customer messages → You reply with catalog/link
2. Customer places order on website
3. Send order confirmation on WhatsApp
4. Send shipping update with tracking
5. After delivery → send review request
6. 30 days later → send new recommendations

### Message Templates for Each Stage:

**Order Received:**
```
✅ Order Confirmed!

Hi [Name]! Your order has been placed successfully.

🧾 Order #[ID]
📦 Items: [list]
💰 Total: ₹[amount]
🚚 Delivery: 3-5 business days

We'll notify you once shipped. Thank you! 🙏
```

**Shipped:**
```
🚚 Your Order is On Its Way!

Hi [Name]! Great news — your order has been shipped.

📦 Order #[ID]
🔗 Track: [tracking_url]
📅 Expected: [date]

Excited for [child's name] to wear it! ✨
```

**Delivered:**
```
📦 Delivered!

Hi [Name]! Your Indistylex order has been delivered.

Hope your little one loves it! 💕

If everything looks great, we'd love a quick review:
⭐ [Google Review Link]

```
