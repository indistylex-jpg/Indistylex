## WhatsApp API (Scale Up — Month 3+)

When you outgrow the basic app, use **WhatsApp Business API** via:
- **Interakt** (₹999/month) — best for Indian D2C brands
- **Wati** (₹2,499/month) — more features
- **AiSensy** (₹999/month) — good automation

### Features you unlock:
- Automated order notifications
- Abandoned cart recovery (auto WhatsApp)
- Broadcast to 10,000+ users
- Chatbot flows
- Green tick verification
