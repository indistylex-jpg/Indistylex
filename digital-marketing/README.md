# Indistylex — Digital Marketing Playbook

> Complete marketing setup for growing a kids' clothing brand online in India.

## 📁 Directory Structure

```
digital-marketing/
├── 01-STRATEGY.md          → Overall marketing plan & budget
├── 02-BRAND-GUIDE.md       → Brand voice, colors, tone
├── 03-SEO-CHECKLIST.md     → On-page SEO, keywords, technical SEO
├── 04-GOOGLE-ADS.md        → Search & Shopping Ads setup
├── 05-META-ADS.md          → Facebook & Instagram Ads
├── 06-SOCIAL-MEDIA.md      → Content calendar & post ideas
├── 07-EMAIL-MARKETING.md   → Automated flows & campaigns
├── 08-WHATSAPP.md          → WhatsApp Business setup
├── 09-INFLUENCER.md        → Outreach templates & strategy
├── 10-ANALYTICS.md         → Tracking setup & KPIs
└── templates/              → Ready-to-use ad copies & creatives brief
```

## 🚀 Quick Start (Do This First)

1. **Week 1**: Set up Google Analytics 4 + Meta Pixel (see `10-ANALYTICS.md`)
2. **Week 1**: Claim Google My Business listing
3. **Week 1**: Set up Instagram Business + Facebook Page
4. **Week 2**: Start posting daily on Instagram (see `06-SOCIAL-MEDIA.md`)
5. **Week 2**: Set up email automation (see `07-EMAIL-MARKETING.md`)
6. **Week 3**: Launch first Meta Ads campaign (see `05-META-ADS.md`)
7. **Week 4**: Start Google Shopping Ads (see `04-GOOGLE-ADS.md`)
